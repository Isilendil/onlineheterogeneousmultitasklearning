This Matlab package implements the algorithm in the following paper:

Yin Zhu, Zhongqi Lu, Sinno J. Pan, Gui-Rong Xue, Yong Yu and Qiang Yang. Heterogeneous Transfer Learning for Image Classification. In Proceedings of the Twenty-Fifth AAAI Conference on Artificial Intelligence (AAAI 2011), San Francisco, California, USA, August 7-11, 2011. 


It is based on a CMF package implemented by Ajit Paul Singh and modified by Nathan Liu. 

The package is implemented by Yin Zhu and is now maintained by Zhongqi Lu. Contacts for Yin and Zhongqi:
